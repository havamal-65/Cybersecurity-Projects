# Geolocation Guesser from Images

Analyzes public images (e.g., from X posts) for clues like landmarks, signs, or metadata to guess where they were taken.

**Hireability:** Highlights image analysis and geolocation skills, key for investigations or threat intel (e.g., locating a suspect).
**Vibe Coding Twist:** Use AI to extract EXIF data or describe image contents (e.g., "I see a street sign in French").
**Tools:** Python (Pillow for images, exif for metadata), or integrate Google Vision API for object recognition.
